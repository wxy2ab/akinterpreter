## `help`

List available commands.

## `clear_history`

Clear chat history.

## `go`

Execute the current plan.

## `reset`

Reset the current state. Restart the application afterward.

## `modify_step_code=index query`

Revise generated code for a specific step using a natural-language request.

## `redo`

Run the current code again without regenerating it unless an error occurs.
