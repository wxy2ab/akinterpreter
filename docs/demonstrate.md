# akinterpreter Features

**English** | [简体中文](./demonstrate.zh-CN.md)

## 1. Generate Code Automatically

akinterpreter generates analysis code from natural-language requests.

![Automatically generated code](./image/auto_gen_code.png)

## 2. Retrieve Data Automatically

Financial analysis often starts with the difficult work of finding the right data. akinterpreter retrieves data for you.

![Automatic data retrieval](./image/auto_get_data.png)
![Automatic data retrieval](./image/auto_retrieval_data.png)

## 3. Repair Generated Code

Generated code can fail. akinterpreter can inspect errors and attempt corrections automatically.

![Automatic code repair](./image/auto_fix_code.png)

## 4. Generate Editable Plans

Before running an analysis, akinterpreter creates a multi-step plan. You can revise the plan until the method matches your goal.

![Automatically generated plan](./image/auto_gen_plan.png)

## 5. Revise Plans Through Chat

You can adjust the plan using natural-language instructions.

![Natural-language plan editing](./image/nlp_editor_plan.png)

## 6. Analyze Data Automatically

akinterpreter selects analysis approaches based on the available data.

![Automatic analysis](./image/data_analysis.png)

The generated analysis can include visualizations.
