# 设置页面

[English](./README.md) | **简体中文**

设置页面是web交易系统的配置中心，用于管理系统的各种配置参数和设置。

## 功能模块

### 1. 登录设置
- **投资者账号**: CTP登录使用的投资者账号
- **密码**: CTP登录密码
- **功能**: 保存登录信息到配置文件，用于后续自动登录

### 2. CTP配置
- **交易服务器地址**: CTP交易服务器的IP和端口（格式：IP:端口）
- **行情服务器地址**: CTP行情服务器的IP和端口（格式：IP:端口）
- **经纪商代码**: 期货公司的经纪商代码
- **应用标识**: CTP应用程序标识
- **授权码**: CTP认证授权码
- **SimNow合约**: SimNow环境的默认合约代码（可选）
- **功能**: 
  - 保存CTP连接参数
  - 测试CTP连接是否正常
  - 验证配置参数格式

### 3. 系统设置
- **界面设置**:
  - 深色主题: 切换明暗主题
  - 自动刷新: 启用/禁用自动数据刷新
  - 语言: 选择界面语言（中文/英文）
  
- **交易设置**:
  - 交易确认: 下单前是否需要确认
  - 声音提醒: 启用/禁用声音提醒
  - 风险等级: 设置风险控制级别（低/中/高）
  
- **日志设置**:
  - 日志级别: 设置日志记录级别（DEBUG/INFO/WARNING/ERROR）
  - 保存日志: 是否保存日志到文件

- **功能**:
  - 导入设置: 从JSON文件导入配置
  - 导出设置: 将当前配置导出为JSON文件
  - 重置设置: 恢复默认设置

### 4. 安全设置
- **会话管理**:
  - 自动登出时间: 设置无操作自动登出时间
  - 记住登录状态: 是否记住登录状态
  
- **密码安全**:
  - 修改密码: 修改登录密码（待实现）
  - 双因子认证: 启用/禁用双因子认证（待实现）

## 技术实现

### 前端组件
- `SettingsPage.tsx`: 主设置页面组件
- `settingsService.ts`: 设置相关的API服务
- `settings.ts`: TypeScript类型定义

### 后端API
- `/api/config/config`: 获取和保存配置
- `/api/config/login-info`: 获取登录信息
- `/api/config/ctp-config`: 获取CTP配置
- `/api/system/test-ctp`: 测试CTP连接
- `/api/system/status`: 获取系统状态
- `/api/system/stop-ctp`: 停止CTP服务

### 数据存储
- **服务器配置**: 存储在 `gui/setting.ini` 文件中
- **应用设置**: 存储在浏览器的 localStorage 中
- **SimNow配置**: 从根目录的 `setting.ini` 文件读取

## 使用说明

### 首次配置
1. 进入"登录设置"标签页，填入投资者账号和密码
2. 切换到"CTP配置"标签页，填入所有必需的连接参数
3. 点击"测试连接"按钮验证配置是否正确
4. 保存配置后，可以开始使用交易功能

### 日常使用
1. 在"系统设置"中调整界面偏好和交易设置
2. 在"安全设置"中管理登录安全相关选项
3. 使用导入/导出功能备份或恢复设置

### 故障排除
1. 如果CTP连接测试失败，请检查：
   - 服务器地址格式是否正确（IP:端口）
   - 网络连接是否正常
   - 经纪商参数是否正确
   
2. 如果设置保存失败，请检查：
   - 浏览器是否允许localStorage
   - 服务器是否正常运行
   - 配置文件权限是否正确

## 开发说明

### 添加新的设置项
1. 在 `types/settings.ts` 中添加类型定义
2. 在 `settingsService.ts` 中添加相关的API调用
3. 在 `SettingsPage.tsx` 中添加UI组件
4. 在后端API中添加对应的处理逻辑

### 扩展功能
- 可以添加更多的主题选项
- 可以实现更复杂的权限管理
- 可以添加配置验证规则
- 可以实现配置的云端同步
